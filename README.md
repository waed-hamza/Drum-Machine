# Drum Machine App

## By: Waed Hafitha.


## About Drum Machine:
This app was built using react hook functional components. It consists of drum pads, and you can play the sounds by clicking on the pads or by your keyboard.


<br>

 ```

 - cd Drum-Machine
 - code .
 ```
 
2. in VS code :

 ```
 - npm install
 - npm start
 ```
